Every request passing through the routing engine leaves a trace. 
You can inspect these traces directly using the local DevTools 
instance at `localhost:4983`.

Open your browser. Send a request from your client application. 
The complete request lifecycle populates immediately.

![DevTools tracing session](/devtools-session.png)

## Trace Capabilities

DevTools provides absolute visibility into the execution pipeline. 
Each layer of the request is isolated and exposed.
